# n8n-nodes-itop

This package provides community nodes for integrating [iTop](https://www.combodo.com/itop-193) (IT Operational Portal) with [n8n](https://n8n.io/).

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n documentation.

## Credentials

The iTop node requires API credentials to connect to your iTop instance. You'll need:

- **URL**: Your iTop instance URL (e.g., `https://your-itop.example.com`)
- **Username**: Your iTop username
- **Password**: Your iTop password

## Supported Operations

### Tickets
- Create ticket
- Get ticket
- Update ticket
- Delete ticket
- List tickets

### Persons
- Create person
- Get person
- Update person
- Delete person
- List persons

### Organizations
- Create organization
- Get organization
- Update organization
- Delete organization
- List organizations
